# Urban Routes Test Automation

## Descripción del Proyecto
Este proyecto automatiza la selección de tarifas y la interacción con la aplicación **Urban Routes** para realizar pruebas automatizadas del proceso de solicitar un taxi. Incluye la automatización de diferentes etapas del flujo de trabajo, como la selección de tarifas, la introducción de un número de teléfono, la validación del código de confirmación, y la adición de una tarjeta de pago.

## Tecnologías y Técnicas Utilizadas
- Lenguaje: Python
- Framework: Selenium WebDriver para automatización de pruebas en navegadores.
- Gestión de Esperas: Se utiliza `WebDriverWait` para gestionar esperas explícitas en los elementos.
- Diseño del Proyecto: Page Object Model (POM) para organizar mejor el código.

## Instrucciones para Ejecutar las Pruebas

1. Instalación de Python
2. Instalación de Selenium
3. Instalación de Pytest
4. Instalación de Chrome
5. Navega al directorio principal del proyecto.
6. Ejecuta: python main.py
